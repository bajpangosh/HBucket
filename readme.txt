A WordPress plugin that offloads media files to Hetzner's S3-compatible Object Storage. Developed by Bajpan Gosh, KLOUDBOY TECHNOLOGIES LLP.

== Description ==

H Bucket offloads media files to Hetzner's S3-compatible Object Storage, rewrites URLs, and offers tools for migration, optimization, and control â€” boosting speed and reducing server load.

**Key Features:**
* Automatically offload media to Hetzner upon upload.
* Rewrite URLs to serve directly from Hetzner.
* Intuitive admin controls for setup and monitoring.
* Support bulk migration of existing media.
* Developer hooks and CLI tools for customization.

== Installation ==

1. Upload the `h-bucket` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Hetzner Offload to configure the plugin.
